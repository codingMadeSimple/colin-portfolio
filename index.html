Charcoal & Cabernet is a web application built for restaurants that allows users to reserve specific seating while booking a reservation. A user can register an account and log in/out. A logged in user can store their favorite dishes between sessions, create orders and view previous orders. After a user places a delivery order they will receive delivery status update texts. If a user would like to dine in they can input a date and time they would like as well as their party size. The user can then select the seat they would like on a map of the restaurant which will show available seats in blue and already booked seats in red.
